Online Shop studentbooks.ch Ein Schweizer Online Shop mit über 18 Mio. Artikeln.

Online Shop mit über 18 Mio. Artikel

Auf Studenbooks können über 18 Mio. Bücher gesucht und gekauft werden. Für alle Bücher ist ein Lieferstatus vorhanden, welcher stets aktualisiert wird.

User können gebrauchte Bücher verkaufen

Durch die Eingabe von ISBN-Nummer können User eigene, gebrauchte Bücher verkaufen. Anhand der ISBN-Nummer werden Informationen über das Buch geladen, die User ergänzen mit Informationen über Zustand sowie Verkaufspreis etc.

Unterlagen von Fachhochschulen und Universitäten

Über die Eingabe der Fachhochschule oder Universität kann nach spezifischen Unterlagen von Dozenten und Schulen gesucht werden.

Ausgangslage

Mit hohen Nutzerzahlen im Hintergrund steht eine gute Skalierung und eine starke Suche im Fokus.

Speziell auf Studenten ausgerichtet stellt Studentbooks ein Onlineshop für verschiedene Artikel, speziell Bücher, zur Verfügung.

Um den administrativen Aufwand zu minimieren, sind

  • eine Anbindung an verschiedene Artikel Datenquellen für den automatischen Datenabgleich
  • eine Anbindung an ein ERP/CRM System für Bestellungen
nötig.

Speziell während der Bestellphase ist es wichtig, einen unterbruchsfreien Betrieb zu gewährleisten.

Produkt

Die Artikel im Shop werden von verschiedenen Datenquellen täglich automatisch aktualisiert, damit immer die aktuellsten Produkte und Lieferstati angezeigt werden können. Dabei werden sowohl das ERP von Studentbooks sowie ein Drittanbieter miteinbezogen.

Darüber hinaus müssen > 18 Mio. Artikel indexiert und per Volltextsuche schnell durchsucht werden können, während die Serverkosten tief gehalten werden müssen.

Für das Hosting managed die Renuo einen Linux Server, der von einem Drittanbieter physisch bereitgestellt wird. Um den Service zu überwachen, wird ein Überwachungstool von NewRelic eingesetzt. Auf diese Weise wird die Renuo sofort informiert, falls etwas nicht stimmen sollte, und kann entsprechend sofort reagieren.
