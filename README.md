## Requirements

* nvm <https://github.com/creationix/nvm>
* nodejs
* npm

## Installation

```sh
git clone https://github.com/renuo/beta.renuo.ch.git && cd beta.renuo.ch
`bin/setup`
```

## Run

`bin/run`

## Check / "Tests"

`bin/check`

## Build / Generate Website

`docpad generate`
